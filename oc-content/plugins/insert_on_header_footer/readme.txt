== Description ==
Insert on Header & Footer allows you to add extra script on your site header and footer.

If you want to learn more about this plugin then we would suggest visiting [DopeThemes / Insert on Header & Footer](http://www.dopethemes.com/plugins/insert-on-header-footer/ "DopeThemes / Insert on Header & Footer").

== Installation ==
1. Upload `insert_on_header_footer/` to the `/oc-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in Osclass
3. Sit back and enjoy


== Frequently Asked Questions ==
1. Q: What's the use of this plugin?
A: Basically this simply allows you to add code inside head and footer, which is helpful for some instances for example you need to add some tracking or verification code like Bing verification code, Google Webmaster verification code, Google Analytics or many more.
